# 💻 Настройка среды разработки

## 🛠 Обязательный софт
- **VS Code**: Основной редактор.
- **Git**: Для синхронизации (см. [[Git-Guide]]).
- **Node.js**: Для запуска локального сервера PWA.
- **Python 3.10+**: Для логики Telegram-бота.

## 🔌 Плагины VS Code
- **Live Server**: Чтобы видеть изменения в HTML/CSS в реальном времени.
- **Prettier**: Чтобы код всегда был ровным и красивым.
- **ESLint**: Чтобы ловить ошибки в JavaScript на лету.

## 🔐 Доступы
- **GitHub**: Убедись, что твой SSH-ключ добавлен в профиль.
- **Vercel/Netlify**: Для деплоя фронтенда.
- **BotFather**: Для получения токена тестового бота.
