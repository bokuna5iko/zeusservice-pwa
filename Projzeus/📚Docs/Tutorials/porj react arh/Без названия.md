- **`src/styles/base.css`**: Здесь только обнуление (`margin: 0`, `box-sizing: border-box`) и настройки шрифтов.
    
- **`src/app.css`**: Здесь настройки того самого «телефона» (ширина 400px, центрирование по центру экрана).
    
- **`src/pages/Login/LoginPage.css`**: Стили только для кнопок и полей ввода экрана логина.


src/
├── main.jsx          # Точка входа: "Заводит" React и подключает самые базовые стили.
├── App.jsx           # Контроллер: Решает, какой экран показать (Логин или Админка).
├── App.css           # Каркас: Стили "оболочки" (фон, центрирование мобильного окна).
│
├── styles/
│   └── base.css      # Глобальный сброс: Шрифты, обнуление отступов (margin: 0).
│
├── pages/            # Папки с конкретными экранами
│   └── Login/
│       ├── LoginPage.jsx   # Только структура экрана логина
│       └── LoginPage.css   # Только дизайн кнопок и полей ввода логина
│
└── components/       # Общие элементы (кнопки, модалки), если они нужны везде
