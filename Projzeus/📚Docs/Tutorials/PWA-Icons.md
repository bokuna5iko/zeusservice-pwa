# 🎨 Работа с иконками PWA

Чтобы приложение выглядело как родное на смартфоне, нужны иконки правильных размеров.

## 📏 Размеры (PNG)
- `192x192 px`: Для меню многозадачности.
- `512x512 px`: Основная иконка при установке.
- `apple-touch-icon`: 180x180 px для iPhone.

## ⚙️ Как обновить
1. Положи новые иконки в папку `/assets/icons/`.
2. Проверь пути в файле `manifest.json`.
3. Убедись, что фон иконок не прозрачный (для Android лучше использовать сплошной фон).

> **Совет:** Используй сервис [PWA Manifest Generator], чтобы быстро нарезать картинки.
