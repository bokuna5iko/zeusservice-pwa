# 📖 Как пользоваться системой

## 📱 Для клиента (PWA)
1. Открой ссылку в браузере Safari (iOS) или Chrome (Android).
2. Нажми "Поделиться" -> "На экран Домой".
3. Теперь приложение запускается с рабочего стола без лишних кнопок браузера.

## 🤖 Для админа (Telegram)
1. Запусти бота командой `/start`.
2. Используй кнопки меню для проверки статуса клиента.
