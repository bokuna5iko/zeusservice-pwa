#### 1. 🌌 Star of Death (Звезда Смерти)

- **Что это:** Глобальный визуальный чертеж всего проекта.
    
- **Как используется:** Здесь хранятся все орбиты, связи и самые смелые идеи. Это «карта мечты».
    
- **Интеграция:** Из орбит звезды ведут ссылки на конкретные файлы в `Readme` и `Docs`.
    

#### 2. 🗺 Roadmap (Карта)

- **Что это:** Тактический план на текущий момент.
    
- **Как используется:** Разбит на папки по версиям (`v0.1`, `v0.2`). В каждой — свой `map.canvas`.
    
- **Интеграция:** Связывает цели из `v0.1` с задачами в **YouGile**.
    

#### 3. 🧠 Readme (Мозг)

- **Что это:** Технический паспорт проекта.
    
- **Как используется:** Описывает структуру модулей (Backend, БД, API). Не «как делать», а «что это такое».
    
- **Интеграция:** Служит промежуточным звеном между визуальной картой и реальным кодом.
    

#### 4. 📱 Progressive Web App (Код)

- **Что это:** Твоя основная разработка.
    
- **Как используется:** Здесь живет весь фронтенд, скрипты, манифесты и стили.
    
- **Интеграция:** Синхронизируется через **Git** с GitHub. Это то, что в итоге увидит пользователь и босс.
    

#### 5. 📚 Docs (Инструкции)

- **Что это:** Твой учебник и база знаний.
    
- **Как используется:** Содержит гайды по настройке (`Setup`), туториалы и планы презентаций.
    
- **Интеграция:** Сюда мы заглядываем, когда забыли команду Git или когда нужно подготовить отчет для босса.
    

#### 6. 📥 Backlog (Инкубатор)

- **Что это:** Свалка и фильтр идей.
    
- **Как используется:** `0-Inbox` для быстрого сброса мыслей, `1-Refined` для тех, что пойдут в следующую версию.
    
- **Интеграция:** Питает новыми задачами `Roadmap` и «Звезду Смерти».
