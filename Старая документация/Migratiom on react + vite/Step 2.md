### 📑 Итоговая сводка миграции Zeus Auto (React + Vite)

**1. Текущая архитектура и стек:**

- **Фронтенд:** React 19 + Vite. Использование `index.html` в корне как точки входа (подключает `src/main.jsx`).
    
- **Компоненты:** Проект полностью разбит на модули (`PointsGrid`, `Navigation`, `History`, `Profile`, `Admin`, `Login`).
    
- **Стили:** Все оригинальные CSS-файлы (`base`, `layout`, `pages`, `components`) интегрированы. В `App.jsx` настроены относительные пути для их импорта.
    
- **PWA:** Полная конфигурация в `vite.config.js` через `vite-plugin-pwa` (манифест, иконки, стратегия `autoUpdate`).
    

**2. Логика и состояние:**

- **AuthContext:** Реализовано глобальное состояние авторизации. Приложение проверяет `accessToken` в `localStorage` при загрузке.
    
- **Навигация:** Интерактивный Tab Bar управляет состоянием `activePage`, переключая экраны без перезагрузки страницы.
    
- **API:** Установлен `Axios` для связи с бэкендом.
